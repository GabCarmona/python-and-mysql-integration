Para Rodar o programa é necessário:
- Python instalado;
- Instalar o MySQL pelo Xampp;
- Instalar o MySQL Workbench;
- Iniciar o MySQL pelo Xampp, abrir a conexão do Workbench e nele rodar os arquivos de criação de script (Script.sql) e o de insert (insertgeral.sql);
- Alterar as configurações da conexão no arquivo Connection3.py;
- Instalar as bibliotecas do python: pandas, sqlalchemy, my-sql-connector, matplotlib
